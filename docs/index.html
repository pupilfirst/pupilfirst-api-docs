Pupilfirst LMS API Reference

This is the documentation for GraphQL queries and mutations that are currently available to clients that authenticate using an API token.

API Endpoints
Development:
http://school.localhost/graphql
Production:
https://my-school-name.pflms.com

Fetch Courses

Request Content-Types: application/json
Query

  courses{

    enableLeaderboard

    featured
    id
    name
    progressionBehavior
    progressionLimit
    publicSignup

Fetch Student Submissions

Loads student submissions for reviewed targets from a course. This is a paginated query.

after:
string

Returns the elements in the list that come after the specified cursor.

before:
string

Returns the elements in the list that come before the specified cursor.

first:
integer

Returns the first n elements from the list.

last:
integer

Returns the last n elements from the list.

courseId:
string

(no description)

status:
SubmissionStatus

(no description)

sortDirection:
SortDirection

(no description)

sortCriterion:
SubmissionSortCriterion

(no description)

levelId:
string

(no description)

coachId:
string

(no description)

Example

Request Content-Types: application/json
Query
query submissions($after: String, $before: String, $first: Int, $last: Int, $courseId: ID!, $status: SubmissionStatus!, $sortDirection: SortDirection!, $sortCriterion: SubmissionSortCriterion!, $levelId: ID, $coachId: ID){
  submissions(after: $after, before: $before, first: $first, last: $last, courseId: $courseId, status: $status, sortDirection: $sortDirection, sortCriterion: $sortCriterion, levelId: $levelId, coachId: $coachId){
    totalCount
  }
}

  "before": "string",
